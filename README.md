# TaskTimer

A Task Management app with time saving features like timer, pomodromo, procrastination killer suite which contains CBT based SMART tasks, Random Tasks Selector, Meditations, Presence Tracker.

## Links

* [Website](https://tasktime27.netlify.app/)
* [Github](https://github.com/moodynooby/TaskTimer)
* [Sister Project](https://github.com/moodynooby/Todo.Txt)

## Features

*   Timer
*   Pomodoro
*   Procrastination Killer Suite
    *   CBT based SMART tasks
    *   Random Tasks Selector
    *   Meditations
    *   Presence Tracker
